# Finance Tracker

A mobile-first personal finance and budget tracking web application.

## Features
- Track income and expenses
- Set budget limits
- Savings goals
- Monthly/yearly reports
- Works offline (PWA)
- Philippine Peso currency

## Deployment
This is a static HTML/CSS/JavaScript application. No build process required.

## Usage
Open `index.html` in a web browser or deploy to any static hosting service.
